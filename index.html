🏗️ Architektur

🌐 Frontend: simplymindo.de

Next.js 14 (App Router), Tailwind CSS. Cloudflare Pages (simplymindo-portal). GitHub: oechteli/simplymindo-portal.

🗄️ Datenbank: Supabase

PostgreSQL via Supabase. Tabellen: profiles, user_apps. RLS-Policies, Auth-Trigger. Schema-Datei: supabase-schema-simplymindo.sql im Repo.

💳 Zahlungen: Stripe

3 Produkte (Einzeln / Flex / Unlimited). Checkout-API + Webhook für automatische App-Freischaltung nach Zahlung.

🤖 KI: Claude Haiku

Anthropic Claude Haiku für alle KI-Funktionen in den Apps. Kostengünstig, schnell, via API angebunden.
