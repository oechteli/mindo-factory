    Du
  2. 3 Stripe-Produkte anlegen Stripe Dashboard → Products → Create. Einzeln: 1,99 € einmalig. Flex: 7,99 €/Monat. Unlimited: 14,99 €/Monat. Die drei Price-IDs notieren.
    Du
  3. Price-IDs in .env.local eintragen Datei: C:\Users\mail\simplymindo\.env.local. Variablen: STRIPE_PRICE_ID_EINZELN, STRIPE_PRICE_ID_FLEX, STRIPE_PRICE_ID_UNLIMITED.
    Du
  4. Cloudflare Pages — Portal verbinden Cloudflare Dashboard → Pages → Connect to Git → Repo oechteli/simplymindo-portal. Framework: Next.js. Build: npm run build. Alle .env-Variablen eintragen.
    Du
  5. Stripe Webhook eintragen Stripe Dashboard → Webhooks → Add endpoint. URL: https://simplymindo-portal.pages.dev/api/webhook. Events: checkout.session.completed, customer.subscription.updated, customer.subscription.deleted. Webhook-Secret in .env.local als STRIPE_WEBHOOK_SECRET.
    Du

    Du
